Me recibí de ingeniero químico en la UNLP pero eso no estaría funcionando como carrera profesional. Acá está mi resume y curriculum si interesa a alguien.

Ya no recuerdo como fue que llegué a este libro pero le hubiera sacado mas provecho después de tomar un curso de CS. Habiendo dicho esto me parece un gran libro en cuanto te manda a buscar info por vos mismo, y a setear tu propio entorno de trabajo, a contramano de muchos libros o cursos que son muy hand-holding. La parte de desarrollo web que lei me resultó muy confusa y ahí es cuando dejé el libro. Nota: la edición que leí era de python 2.7. Hay ediciones mas recientes que tratan de python 3.

Codecademy te va a poner en contacto con un monton de lenguajes distintos de manera superficial, sin muchas chances de crear grandes proyectos, al menos en su versión gratuita. En este curso practiqué python, javascript, react y jquery. La parte de SQL tiene un salto muy grande de dificultad al inicio, pero debería volver con mas atención para plantear las query. Repito que en su versión gratuita sirve para conocer un lenguaje ya sabiendo otro.

Angela Yu brinda uno de los mejores cursos que tomé (fuera y dentro de la informática). Su curso de desarrollo web es completo, frontend y backend, llevando a cabo una serie importante de proyectitos y tiene una gran lista de recursos para leer. Esto conlleva una carga horaria importante (alrededor de 40 hs). Cuando tomé este curso se alentaba el uso de node, express y mongodb para las bases de datos. Creo que con los conocimientos de ahora le podría sacar mas provecho.

Este libro esta gratis, y la edición que leí es un poco vieja (esta basado en python 2.7 y tiene algunas cosas descontinuadas. Tiene un repaso bastante completo por todas la caracteristicas de python, estructuras de datos y debugging. Por ahí le falta algo de algoritmos para ser totalmente completo. El autor cuenta con una serie de libros (estadisticas y java creo que estan) de este estilo que pienso revisar.

Este curso es ideal, casi perfecto, para aquellas personas (me incluyo) que necesitan una base de fundamentos teóricos para sacar el máximo provecho de las cosas. Dicho esto, entiendo que este curso puede ser dificil para gente que viene con cero background de programación. El curso está planteado para durar un semestres, pero dedicandose, se puede terminar en aproximadamente un mes y medio (si se tiene la honestidad académica que pregona el curso). Empieza con C, y en la segunda mitad sigue con python, SQL, y HTML-CSS-Javascript. Cuando hayan transcurrido las primeras seis semanas vas a haber cocinado un par de programas muy interesantes.

También hice un par de cursos mas que no creo que me hayan dado buenos fundamentos, entre ellos uno de Machine Learning dictado por un español en Udemy y la plataforma Khan Academy.

Si tuviera que recomendar un orden de este material que he leido. Sería:

  1. Think Python: How to think like a computer scientist
  2. CS50
  3. Codecademy
  4. Web Development Bootcamp de Angela Yu
  5. Learn Python the hard way
